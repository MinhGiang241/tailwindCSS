Tailwind css for beginner

tailwind is a utility CSS framework. In contrast to the other CSS framework like Bootstrap or Materialize CSS it doesn't come with predefined components. Instead TailwindCSS operates on a lower level and provides you with a set of CSS helper classes. By using this classes you can rapidly create custom design with ease TailwindCSS is not opinionated and let's you create your own unique design
